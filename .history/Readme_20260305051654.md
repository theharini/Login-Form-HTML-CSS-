
# Login Form Project

## Overview
This is a simple and responsive **Login Form** built using **HTML** and **CSS**.  
It features a centered login box with input fields for user credentials and a stylish login button. The design is mobile-friendly and uses modern styling elements like rounded corners, hover effects, and a subtle box shadow.

---

## Features
- Responsive login form centered both vertically and horizontally.
- Stylish input fields with rounded corners.
- Hover effect on the login button for better user interaction.
- Clean and minimalistic design using only HTML and CSS.
- Easy to customize colors, fonts, and sizes.

---

## Technologies Used
- HTML5
- CSS3
